## Project Overview:
The project is named **Picturedom** and the idea is to create something resembling **Instagram** or **Pinterest**. Mainly photo-oriented website.

## Main functionality:
#### Images:
- can be liked and disliked
- can be viewed
- can be posted
- can be commented on

#### Comments:
- can be liked and disliked
